COLABORAÇÃO PARA A FOLHA
Para ter acesso ao banco de dados do StumbleUpon, é preciso instalar uma barra de ferramentas no computador _os criadores do site garantem que o usuário não terá surpresas ruins com softwares espiões.
Clique no link Sign-Up Now e diga qual sistema operacional usa. Imediatamente, o download da barra de ferramentas será efetuado. São 282 Kbytes, baixados em poucos segundos.
Ao fim do download, uma barra de ferramentas é adicionada à janela do navegador (além do Internet Explorer, o StumbleUpon funciona com o Firefox, o Mozilla e o Netscape). O próximo passo é criar uma conta para participar da comunidade. Basta clicar em Welcome to StumbleUpon!.
O usuário deve escolher um apelido e informar a cidade de origem, a idade e o sexo. Uma lista de temas está disponível para que ele escolha seus preferidos. No fim da página, é só digitar um endereço de e-mail para receber a senha de acesso.
A opção Menu, na barra de ferramentas, leva o usuário ao Sign-in. Depois de clicar nele, digite o nome escolhido e a senha. Aí é só procurar por outros usuários para saber o que eles têm a indicar.
Se preferir, o botão Stumble! oferece sugestões de sites. O sistema leva em consideração as preferências definidas pelo internauta, que podem ser modificadas.
Sempre que você visitar um site, poderá fazer uma resenha e/ou classificá-lo, dizendo se gosta (I like it!) ou não (Not for me) do endereço. Outra opção é indicar sites diretamente para outros participantes _para isso, é preciso saber o nome de cadastro deles.
